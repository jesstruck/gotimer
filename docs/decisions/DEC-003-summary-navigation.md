# DEC-003: Summary Type and Week Navigation UX

- Status: Accepted
- Date: 2026-03-14
- Type: UX/Product

## Decision
- Daily summary is excluded from MVP.
- Weekly and monthly summaries are included in MVP.
- No generic filtering UI is included in MVP.
- User must be able to move backward/forward across weeks with one-click actions.
- Week boundaries follow ISO week (Monday-Sunday).

## Impact
- UI must prioritize simple navigation controls over advanced filter controls.
- Backend summary endpoints must support week and month summary retrieval.
